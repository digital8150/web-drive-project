# 📂 LiteDrive 실행 가이드 (HOWTO)

이 문서는 LiteDrive 프로젝트를 로컬 환경에서 설치하고 실행하는 방법을 안내합니다.

## 1. 필수 환경
- **Node.js**: v14 이상 권장
- **npm**: Node.js 설치 시 함께 설치됩니다.

## 2. 의존성 설치
프로젝트 루트 디렉토리에서 아래 명령어를 실행하여 필요한 패키지를 설치합니다.

```bash
npm install
```

## 3. 환경 변수 설정 (`.env`)
프로젝트 루트에 `.env` 파일을 생성하고 아래 내용을 입력합니다. (이미 생성되어 있을 수 있습니다.)

```env
PORT=3000
DATA_PATH=./uploads
ADMIN_PASSWORD=admin123
```
- `PORT`: 서버가 실행될 포트 번호입니다.
- `DATA_PATH`: 파일이 저장 및 공유될 로컬 경로입니다.
- `ADMIN_PASSWORD`: 파일 업로드를 위한 관리자 비밀번호입니다.

## 4. 서버 실행

### 운영 모드 (Production)
```bash
npm start
```

### 개발 모드 (Development - nodemon 사용)
```bash
npm run dev
```

서버가 실행되면 터미널에 아래와 같은 로그가 출력됩니다.
`LiteDrive Server is running on http://localhost:3000`

## 5. 접속 및 기능 사용

### 🌐 접속
브라우저에서 `http://localhost:3000`으로 접속합니다.

### 📂 주요 기능 사용법
1. **파일 탐색**: 메인 화면에서 `uploads` 폴더 내의 파일 리스트를 확인할 수 있습니다.
2. **미리보기**: 이미지, 비디오(MP4), PDF, 텍스트 파일은 'View' 버튼을 눌러 브라우저 내에서 즉시 확인할 수 있습니다.
3. **다운로드**: 파일명을 클릭하거나 'Download' 버튼을 눌러 파일을 내려받습니다.
4. **관리자 모드 및 업로드**:
   - 상단 'Admin Password' 입력란에 `.env`에 설정한 비밀번호(기본: `admin123`)를 입력하고 'Login'을 누릅니다.
   - 로그인 성공 시 업로드 섹션이 나타나며, 파일을 선택하여 현재 탐색 중인 경로로 업로드할 수 있습니다.

## ⚠️ 주의 사항
- **보안**: 상위 디렉토리 접근(Path Traversal)은 서버 단에서 차단되어 있으나, 공유할 폴더(`DATA_PATH`) 설정 시 주의하십시오.
- **용량 제한**: 기본 업로드 용량 제한은 **100MB**로 설정되어 있습니다. (설정 변경 시 `src/index.js` 수정 필요)
