# 📂 LiteDrive: Node.js 기반 초간편 웹 드라이브

이 프로젝트는 서버 디스크의 파일을 HTTP를 통해 공유하고 미리보기 및 업로드를 지원하는 경량 웹 드라이브 서버입니다.

## 🛠 기술 스택
- **Runtime**: Node.js
- **Framework**: Express.js
- **UI**: HTML5, CSS3, JavaScript
- **Auth**: JWT/Session
- **VCS**: Git
